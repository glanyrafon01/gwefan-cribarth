Cribarth Music Player

Mae Cribarth Music Player yn rhaglen bwrdd gwaith fach sy’n galluogi Music Assistant yn Home Assistant i dargedu’r ddyfais leol fel pwynt chwarae parhaus, yn annibynnol o’r porwr. Mae wedi’i chynllunio ar gyfer symlrwydd, sefydlogrwydd, ac integreiddio gyda lansiwr y system.

Nodweddion

  • Chwaraewr Electron sy’n lapio nodwedd “This Device” Music Assistant
  • Lansiwr bwrdd gwaith a eicon personol
  • Wedi’i ddosbarthu fel pecyn .deb ar gyfer Linux a .exe ar gyfer Windows dyfeisiau
  • Gosod a diweddaru a llaw er symlrwydd

Sut cafodd ei adeiladu

  • Datblygwyd yn Visual Studio Code
  • Darparodd ChatGPT (GPT-4) y prif ganllaw datblygu, gan gynnwys strategaeth pecynnu, integreiddio eicon, a chymorth dadfygio
  • Defnyddiwyd GitHub Copilot i gyflymu golygu ar lefel cystrawen
  • Fersiynwyd, adeiladwyd, a rhyddhawyd trwy GitHub Actions a rhyddiadau GitHub wedi’u tagio

Cymwysiadau

  • Chwarae lleol ar gyfer Music Assistant heb angen porwr ar agor
  • Integreiddio sain aml-ystafell parhaus
  • Chwaraewr cerddoriaeth ysgafn mewn amgylcheddau fel ciosgiau neu ddesgben ysgafn
